    1. Multiple transitiosn can be inlined into a singular statement:
      transition 0.2s ease-out 50ms;
    2. Keyframe: a snapshot that defines a starting or ending point of any smooth transition. Key terms are "from" and "to" (0% to 100% respectively). They specifiy how far along the animation each keyframe is located. Use @keyframes rule in CSS.
    3. Canvas: Draw directly on the screen which speeds up performance - as opposed to downloading images. Set the width/height inline and not css.
    4. The coordinate system is different from the Cartesian coords. Upper left is (0,0) and lower right is (x,x).
    5. Can also display and manipulate images and video.
    6. SVG: Scalable Vector Graphics - which is in XML format that describes the vector image.
